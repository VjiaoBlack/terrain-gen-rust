## Design Drift

### ExplorationMap trivially saturated (NEW)
`exploration_pct=100.0` for all 3 evaluation seeds at tick 6000. Root cause: initial reveal
radius=15 at spawn (game/mod.rs:1369) + villager sight_range=22 covers a 70x25 viewport in ~200
ticks. The ExplorationMap exists and is tracked, but fog-of-war is not functioning as a meaningful
game mechanic. Design doc (game_design.md Pillar 1 brainstorm, Pillar 3 core features) envisions
"villagers only see terrain they've visited" as a key driver of early-game exploration. Added to
features.json known_issues for simulation_maps.

all villagers perished, peak pop=4, survived to Spring Y1D6. This is the same condition as golden
seed 42 (pop=4, fragile). Demonstrates that non-golden seeds have meaningful failure risk. The 3
golden seeds are unrepresentative of typical outcomes.
